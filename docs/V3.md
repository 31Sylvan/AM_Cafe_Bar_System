# Coffee Shop OS V3

V3 聚焦上线后的生产运营能力，优先补齐备份、异常治理和持续运营检查。

## 已落地

- 数据备份中心
  - 页面：`/backup`
  - 集中下载库存、流水、产品、损耗、利润、成本、支出、现金流和员工绩效 CSV
  - API：`/api/backup/manifest`
  - 输出备份清单和绝对下载 URL，便于外部脚本或定时任务拉取
- 经营异常中心
  - 页面：`/quality`
  - 总览异常总数、缺配方产品、负库存和低库存
  - 提供补货建议和数据备份快捷入口

## 下一批候选

- 审计日志：记录关键操作、操作者、时间、关联单据和结果。
- 自动化健康检查：周期性检查 Supabase 环境、低库存、负库存和缺配方。
- 数据保留策略：定义导出、归档和月结快照保留周期。
